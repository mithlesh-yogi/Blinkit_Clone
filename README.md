# Blinkit Clone

A responsive web clone of the Blinkit e-commerce platform built with HTML and CSS.

## Overview

This project is a frontend implementation of the Blinkit grocery delivery application interface. It features a modern, responsive design that works seamlessly across different screen sizes.

## Features

- **Responsive Design**: Optimized layout for desktop, tablet, and mobile devices
- **Product Showcase**: Display of various product categories and items
- **Navigation**: User-friendly navigation menu and header
- **Media Queries**: CSS media queries for adaptive styling on different screen sizes
- **Image Assets**: High-quality product images in AVIF format

## Project Structure

```
Blinkit_Clone/
├── index.html           # Main HTML file
├── css/
│   ├── style.css       # Main stylesheet
│   └── media.css       # Media queries for responsive design
├── images/             # Product and UI images
│   ├── box*.avif       # Category box images
│   ├── card-box*.avif  # Product card images
│   ├── product-*.avif  # Individual product images
│   ├── new-product-*.avif # New product images
│   ├── foot-pannel-img-* # Footer panel images
│   └── widge.avif      # Widget image
└── README.md           # This file
```

## Technologies Used

- **HTML5**: Semantic markup for structured content
- **CSS3**: Modern styling with flexbox/grid layouts
- **Responsive Design**: Media queries for mobile-first approach
- **AVIF Images**: Modern image format for optimized file sizes

## How to Use

1. Open `index.html` in your web browser
2. Navigate through the different sections to view the product categories
3. The layout will automatically adapt based on your screen size

## Getting Started

Simply clone or download this repository and open `index.html` in any modern web browser. No build tools or dependencies required.

## Browser Support

Works on all modern browsers that support:
- HTML5
- CSS3 Flexbox/Grid
- AVIF image format

## Responsive Breakpoints

The design includes media queries to handle:
- Mobile devices
- Tablets
- Desktop screens

Check `css/media.css` for specific breakpoint configurations.
